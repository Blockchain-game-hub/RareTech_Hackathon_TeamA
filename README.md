## 概要
「OSI参照モデル」、「クリーンアーキテクチャー」、「三層アーキテクチャー」などITの知識には図で説明するような事が多い。
自身と勉強仲間の知識を深めるため実際に図を書くというアウトプットを行い、ゲームをしながら勉強ができる。
